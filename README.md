# E-Commerce Platform

A full-stack E-Commerce web application built with **Spring Boot**, **Java**, **Maven**, and **JSP**. Users can register, login, browse products, manage their cart, and complete purchases. Admins can manage products and monitor user activities.

## Features
- User registration/login with email verification
- Product listing, details, and search
- Shopping cart management (add, update, delete items)
- Admin dashboard for product and user management
- JSP-based front-end with dynamic pages

## Project Structure
```
E-commerce/
├── .mvn/                  # Maven wrapper
├── src/
│   ├── main/java/com/MainApp/
│   │   ├── Controller/    # Handles HTTP requests
│   │   ├── Entity/        # Data models
│   │   ├── Repository/    # Database interactions
│   │   └── Service/       # Business logic
│   ├── main/resources/    # Config files
│   └── main/webapp/       # JSP pages & static assets
├── .gitattributes
├── .gitignore
├── mvnw / mvnw.cmd
└── pom.xml
```

## Technologies
Java 17, Spring Boot, Maven, MySQL, JSP, Apache Tomcat
